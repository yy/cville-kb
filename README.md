# Charlottesville Knowledge Base

A community knowledge base focused on Charlottesville, Virginia — city governance, transportation, infrastructure, planning, and civic advocacy.

## What's here

Factual, source-backed information about how Charlottesville works: city council decisions, transportation planning, infrastructure projects, public processes, and how to get involved. Content is organized as an Obsidian vault with interlinked pages.

## Using this vault

This repo is an [Obsidian](https://obsidian.md/) vault. Clone the repo and open it in Obsidian to browse interlinked pages with the graph view. You can also read the Markdown files directly on GitHub.

## Disclaimer

This is a community knowledge base. Content represents contributors' understanding of publicly available information and may contain errors. It is not legal advice and is not affiliated with the City of Charlottesville. Corrections and improvements are welcome.
